I have recently had to create an interactive menu for a project. I started programming it in bash and to tell the truth I quickly felt limited when I wanted to handle more complex cases and especially that I had to add more code from scratch. As the project is built in JavaScript, I told myself that there was probably a package that can do all what I want. After some research, I found it! The package is Inquirer.js. It‚Äôs a collection of common interactive command line user interfaces.

Here is a simple example with a list of choices

// assuming that `inquirer` has already been installed
const inquirer = require("inquirer");

inquirer
  .prompt([
    {
      type: "list",
      name: "prize",
      message: "Select a prize",
      choices: ["cake", "fries"]
    }
  ])
  .then((answers) => { 
    console.log(JSON.stringify(answers, null, "  "));
  });
